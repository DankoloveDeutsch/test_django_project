# Система управления табельным учетом специалиста

## 📋 О проекте

Автоматизированная информационная система для управления кадровым учетом, 
документооборотом и контролем рабочего времени сотрудников.

### Основные возможности

- ✅ Электронные личные дела сотрудников
- ✅ Автоматическое формирование приказов и договоров
- ✅ Система напоминаний о сроках документов
- ✅ Интеграция с бухгалтерскими системами (1С)
- ✅ Электронный табель учета рабочего времени
- ✅ Генерация отчетности для государственных органов
- ✅ Поиск и фильтрация сотрудников
- ✅ Хранение шаблонов документов

### Технологии

- **Backend:** Django 4.2, Python 3.11
- **Database:** PostgreSQL / SQLite
- **API:** Django REST Framework
- **Task Queue:** Celery + Redis
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Document Generation:** ReportLab, python-docx, openpyxl
- **Notifications:** Email, Telegram

## 🚀 Быстрый старт

### Требования

- Python 3.11+
- PostgreSQL 14+ (или SQLite для разработки)
- Redis (для Celery)
